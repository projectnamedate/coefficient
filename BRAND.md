# Coefficient Brand Guide

**Coefficient** is a product by **mythx**, a Solana validator operator and stake pool participant.

---

## Colors

### Primary
| Name | Hex | Usage |
|------|-----|-------|
| Lavender | `#b5b2d9` | Primary brand color, headers, accents |
| Light Lavender | `#d9daed` | Backgrounds, cards, hover states |

### Supporting
| Name | Hex | Usage |
|------|-----|-------|
| Near Black | `#232323` | Text, dark backgrounds, contrast |
| Warm Beige | `#cdc8bd` | Borders, muted elements, secondary backgrounds |

### Secondary (Semantic / Score Colors)
| Name | Hex | Usage |
|------|-----|-------|
| Terra Cotta | `#ae4845` | Bad scores, warnings, sandwich validators |
| Lime Green | `#abd079` | Good scores, healthy metrics |
| Yellow | `#eee56e` | Medium scores, caution |
| Blue | `#4e5fab` | Info, links, neutral data |

---

## Typography

### Primary Font: Rigidly Thoughtfully
- **Usage:** Logo, page titles, hero headings
- **Style:** Blackletter/gothic display font
- **Notes:** Can enlarge single letters and crop to create abstract background shapes

### Secondary Font: Proxima Nova
- **Usage:** Body text, table data, labels, navigation, all UI text
- **Weights:** Regular (400), Medium (500), Semibold (600), Bold (700)

---

## Design Elements

### Texture Overlays
- Use screen blending mode with adjustable opacity
- Apply to hero sections and feature backgrounds

### Letter Abstractions
- Enlarge single characters from Rigidly font
- Crop to create geometric/abstract shapes
- Use as section dividers or background elements

### Eye Patterns
- Small eye/dot motifs from mythx brand mascot
- Use to fill empty space in minimal layouts
